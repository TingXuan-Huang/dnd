**Source:** D&D Beyond Basic Rules (2024)  
**Date:** 2026-05-20 | **Version:** 1.0

Rules beyond a single-class level 1 build: multiclassing, starting at
higher levels, and mixing 2014 Player’s Handbook content with 2024
Basic Rules.

## Multiclassing

Gain levels in a new class when you advance instead of your current
class. XP cost uses **total character level**, not per-class level.

### Prerequisites

**13+** in the primary ability of your **current** class(es) and the
**new** class.

Example: Barbarian → Druid needs Strength 13+ and Wisdom 13+.

### Hit Points and Hit Dice

- Level 1 HP from a class only when **total character level is 1**.
- Later class levels: roll (or average) that class’s Hit Die per level.
- Pool Hit Dice separately if die types differ (e.g. d8 Cleric + d10
  Paladin).

### Proficiency Bonus

Based on **total character level** (see Character Advancement in
`01-creating-a-character.md`).

### Proficiencies Gained

First level in a **new** class grants only the proficiencies listed in
that class’s “As a Multiclass Character” entry (not full starting
proficiencies).

### Stacking Rules

| Feature type | Rule |
| --- | --- |
| Armor Class | Use only one calculation (e.g. Monk Unarmored Defense OR Sorcerer Draconic Resilience) |
| Extra Attack | Does not stack; max two attacks unless a feature says otherwise |
| Spellcasting | See below |

## Multiclass Spellcasting

When you have Spellcasting from **more than one** class:

**Combined spellcaster level**

| Contribution | Formula |
| --- | --- |
| Bard, Cleric, Druid, Sorcerer, Wizard | + full class levels |
| Paladin, Ranger | + half levels (round up) |
| Eldritch Knight, Arcane Trickster | + one-third Fighter/Rogue (round down) |

Look up combined level on the slot table below. Prepare spells **per
class** as if single-classed at that class level. Cantrip scaling uses
**total character level**.

**Pact Magic (Warlock):** Warlock slots can cast other classes’ prepared
spells and Spellcasting slots can cast Warlock spells.

## Starting at Higher Levels

When the DM starts you above level 1:

| Starting Level | Equipment & Money | Magic Items |
| --- | --- | --- |
| 2–4 | Normal starting gear | 1 Common |
| 5–10 | 500 GP + 1d10×25 GP + normal gear | 1 Common, 1 Uncommon |
| 11–16 | 5,000 GP + 1d10×250 GP + normal gear | 2 Common, 3 Uncommon, 1 Rare |
| 17–20 | 20,000 GP + 1d10×250 GP + normal gear | 2 Common, 4 Uncommon, 3 Rare, 1 Very Rare |

- Record subclass at level 3+ on your sheet.
- Use Character Advancement table for XP thresholds and PB.

## 2014 vs 2024 — Older Books on D&D Beyond

The 2024 Basic Rules use a **background + species** origin model:

- **Background** grants an **Origin feat**, skill/tool proficiencies,
  and ability score options (see `02-character-origins.md`).
- **Species** grants traits (Speed, Size, Darkvision, etc.) — not the
  2014 “+2/+1 ability” racial pattern alone.

D&D Beyond still hosts **2014 Player’s Handbook** backgrounds, races,
and subclasses separately from the free **2024 Basic Rules** set.

> When mixing 2014 and 2024 content, use the D&D Beyond character
> builder ruleset your table agrees on (2014 Core vs 2024 Free Rules).
> Select class, background, and species from the matching rules section.
> Do not assume 2014 racial ASI stacks with 2024 background ASI without
> DM approval.

**Practical checklist**

| Topic | 2024 Basic Rules | 2014 PHB (typical) |
| --- | --- | --- |
| Origin | Background + species | Race + background |
| ASI at level 1 | Background + species options | Racial + class at 4/8/… |
| Feats at level 1 | Origin feat from background | Variant human, etc. |
| Weapon mastery | Class feature required | Not in 2014 core |
| Spell lists | 2024 class lists | 2014 lists (overlap differs) |

For official conversion notes when picking a 2014 background or species,
see the **“Backgrounds and Species from Older Books”** sidebar on D&D
Beyond under Creating a Character (anchor:
`BackgroundsandSpeciesfromOlderBooks`).

**Subclass overlap:** Some classes offer alternate subclass options in
older books (e.g. Cleric Divine Order). If you gain the same benefit
twice, use only the option you select for the 2024 feature.
