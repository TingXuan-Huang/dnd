# Creating a Character

**Source:** D&D Beyond Basic Rules (2024)  
**Author:** Wizards of the Coast (compiled)  
**Date:** 2026-05-20 | **Version:** 1.0

---

## Get Ready

Talk with your DM about campaign tone and starting level. Choose a character sheet format (paper, D&D Beyond, or custom).

---



## Create Your Character — Five Steps

1. **Choose a Class**
2. **Determine Origin** (background, species, languages)
3. **Determine Ability Scores**
4. **Choose an Alignment**
5. **Fill in Details** (features, HP, AC, attacks, spells)

---



## Step 1: Choose a Class


| Class     | Likes...       | Primary Ability       | Complexity |
| --------- | -------------- | --------------------- | ---------- |
| Barbarian | Battle         | Strength              | Average    |
| Bard      | Performing     | Charisma              | High       |
| Cleric    | Gods           | Wisdom                | Average    |
| Druid     | Nature         | Wisdom                | High       |
| Fighter   | Weapons        | Strength or Dexterity | Low        |
| Monk      | Unarmed combat | Dexterity and Wisdom  | High       |
| Paladin   | Defense        | Strength and Charisma | Average    |
| Ranger    | Survival       | Dexterity and Wisdom  | Average    |

| Sorcerer  | Power          | Charisma              | High       |
| Warlock   | Occult lore    | Charisma              | High       |

- Write **level** (usually 1) and **XP** (0 at level 1).
- Note **armor training** from your class.

## Step 2: Determine Origin

### Choose a Background


| Background | Ability Scores | Feat                    | Skills                   | Tool                    |
| ---------- | -------------- | ----------------------- | ------------------------ | ----------------------- |
| Acolyte    | Int, Wis, Cha  | Magic Initiate (Cleric) | Insight, Religion        | Calligrapher’s Supplies |
| Criminal   | Dex, Con, Int  | Alert                   | Sleight of Hand, Stealth | Thieves’ Tools          |
| Sage       | Con, Int, Wis  | Magic Initiate (Wizard) | Arcana, History          | Calligrapher’s Supplies |
| Soldier    | Str, Dex, Con  | Savage Attacker         | Athletics, Intimidation  | Gaming Set (choice)     |


Each background: **+2 to one score and +1 to another** from its three listed scores, **or +1 to all three** (max 20).

### Choose Starting Equipment

Take the background package **or** 50 GP. Then add class equipment. You may take **one trinket** (see Trinkets table at end of this file).

### Choose a Species

Dragonborn, Dwarf, Elf, Gnome, Goliath, Halfling, Human, Orc, Tiefling — full traits in `02-character-origins.md`.

### Choose Languages

## Step 4: Choose an Alignment


| Code | Name            | Summary                      |
| ---- | --------------- | ---------------------------- |
| LG   | Lawful Good     | Society’s expected good      |
| NG   | Neutral Good    | Best effort within rules     |
| CG   | Chaotic Good    | Conscience over expectations |
| LN   | Lawful Neutral  | Law, tradition, code         |
| N    | Neutral         | Avoids taking sides          |
| CN   | Chaotic Neutral | Personal freedom             |
| LE   | Lawful Evil     | Orderly selfishness          |

## Step 5: Fill in Details

### Record Class Features

Copy all **level 1** features from your class. Make subclass choices at level 3 unless starting higher.

### Key Numbers


| Item                       | Formula                            |
| -------------------------- | ---------------------------------- |
| Saving throws (proficient) | Ability mod + proficiency          |
| Skills (proficient)        | Ability mod + proficiency          |
| Passive Perception         | 10 + Wis (Perception) modifier     |
| Initiative                 | Dexterity modifier                 |
| Base AC (unarmored)        | 10 + Dex modifier                  |
| Melee attack               | Str mod + proficiency              |
| Ranged attack              | Dex mod + proficiency              |
| Spell save DC              | 8 + spellcasting mod + proficiency |
| Spell attack               | spellcasting mod + proficiency     |


### Level 1 Hit Points


| Class group                               | HP at level 1 |
| ----------------------------------------- | ------------- |
| Barbarian                                 | 12 + Con mod  |
| Fighter, Paladin, Ranger                  | 10 + Con mod  |
| Bard, Cleric, Druid, Monk, Rogue, Warlock | 8 + Con mod   |
| Sorcerer, Wizard                          | 6 + Con mod   |


### Spellcasting at Level 1

If your class has Spellcasting or Pact Magic, record **cantrips known**, **spells prepared**, and **spell slots** from the class table. See `06-spells.md`.

| ----- | ------- | ----------- |

| Class group                               | HP per level |
| ----------------------------------------- | ------------ |
| Barbarian                                 | 7 + Con      |
| Fighter, Paladin, Ranger                  | 6 + Con      |
| Bard, Cleric, Druid, Monk, Rogue, Warlock | 5 + Con      |

### Tiers of Play


| Tier | Levels | Scope                   |
| ---- | ------ | ----------------------- |
| 1    | 1–4    | Local threats           |
| 2    | 5–10   | Regional threats        |
| 3    | 11–16  | Continental threats     |
| 4    | 17–20  | World/multiverse stakes |

## Starting at Higher Levels


| Starting Level | Extra GP                | Magic Items                               |
| -------------- | ----------------------- | ----------------------------------------- |
| 2–4            | Normal gear             | 1 Common                                  |
| 5–10           | 500 GP + 1d10×25 GP     | 1 Common, 1 Uncommon                      |
| 11–16          | 5,000 GP + 1d10×250 GP  | 2 Common, 3 Uncommon, 1 Rare              |
| 17–20          | 20,000 GP + 1d10×250 GP | 2 Common, 4 Uncommon, 3 Rare, 1 Very Rare |


Minimum XP for level = threshold in advancement table. Choose subclass if level 3+.
