# Rules Catalog (granular 5.5e data)

One-page-per-entity reference, formatted close to D&D Beyond entries. Best for
fast **lookup**. For cited, prose explanations of *how rules work*, use
[`../wiki/`](../wiki/) instead (see the root [`README.md`](../README.md) for the split).

| Folder | Contents |
|---|---|
| [`monsters/`](monsters/) | **378 monster stat blocks**, A–Z (the bestiary the wiki lacks) |
| [`equipment/`](equipment/) | Weapons, armor, gear, tools, magic items, mounts, services |
| [`spells/`](spells/) | Spell catalog (casting & gaining) |
| [`classes/`](classes/) | One folder per class |
| [`rules/`](rules/) | Rules glossary (actions, conditions, combat, movement, etc.) |
| [`dm-toolbox/`](dm-toolbox/) | DM reference — traps, poison, curses, environmental effects, encounters |
| [`origin/`](origin/) | Species & backgrounds |
| `class.md`, `skills.md`, `expansion.md` | Top-level indexes; `expansion.md` lists entries that live behind the D&D Beyond paywall |

> Removed during consolidation: the original `references/` folder (808
> auto-generated D&D Beyond link-stubs with no real content) and the
> `.build_references.py` script that produced them.
