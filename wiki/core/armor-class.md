# Armor Class (AC)

Your AC is the target number an [[attack-roll]] must meet or beat to hit you.

**Base AC** (without armor) = 10 + your [[dexterity]] modifier. Armor and certain class features may replace this formula. [BR-2024 §07-playing-the-game#Attack Rolls]

## Common AC formulas

| Source | Formula |
| --- | --- |
| Unarmored (default) | 10 + Dex |
| Light armor | armor's base AC + Dex |
| Medium armor | armor's base AC + Dex (max +2) |
| Heavy armor | armor's base AC (no Dex) |
| Shield | +2 to AC (any of the above) |
| Barbarian Unarmored Defense | 10 + Dex + Con (shield OK) |
| Monk Unarmored Defense | 10 + Dex + Wis (no shield) |

(Class and species features may grant other formulas — see the relevant class page.)
