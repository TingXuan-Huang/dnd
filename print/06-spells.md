# Spells

**Source:** D&D Beyond Basic Rules (2024)  
**Date:** 2026-05-20 | **Version:** 1.0

---

Rules for preparing and casting spells, plus level 0–1 spell name
references for character building. Full spell text is in the Spells
chapter on D&D Beyond.

## Gaining and Preparing Spells

You must have a spell prepared or access via a magic item (e.g. spell
scroll). Your class features define which spells you know or prepare.

| Class | Change list when... | How many change |
| --- | --- | --- |
| Bard | Gain a level | One |
| Cleric | Finish long rest | Any |
| Druid | Finish long rest | Any |
| Paladin | Finish long rest | One |
| Ranger | Finish long rest | One |
| Sorcerer | Gain a level | One |
| Warlock | Gain a level | One |
| Wizard | Finish long rest | Any |

Always-prepared spells (from features) do not count against your
prepared list.

## Casting Spells (Essentials)

### Spell Level and Slots

- Cantrips are level 0 (no slot).
- Casting a level 1+ spell expends a slot of that level or higher.
- **Long rest** restores expended slots.
- **One spell slot per turn** for slot-based spells (Magic action + one
  bonus-slot spell max).

### Casting Without Slots

- Cantrips
- Rituals (+10 min, no slot; must have spell prepared)
- Class features with limited free casts
- Magic items (e.g. spell scroll)

### Upcasting

Higher-level slots can cast lower-level spells; some spells scale in
the description.

### Spell Save DC and Attack

```
Spell save DC = 8 + spellcasting ability mod + Proficiency Bonus

Spell attack = spellcasting ability mod + Proficiency Bonus
```

### Schools of Magic

| School | Typical effects |
| --- | --- |
| Abjuration | Protection, negation |
| Conjuration | Transport, summon |
| Divination | Information |
| Enchantment | Mind influence |
| Evocation | Energy, damage |
| Illusion | Deception |
| Necromancy | Life/death |
| Transmutation | Transformation |

### Components

- **V:** Spoken words; blocked if gagged or in silence
- **S:** Hand gestures
- **M:** Materials (pouch or focus unless consumed/costly)

### Concentration

Some spells require Concentration while active. Broken concentration
ends the spell (see Rules Glossary).

## Multiclass Spellcasting

If you have Spellcasting from more than one class, combine levels:

| Classes | Count as |
| --- | --- |
| Bard, Cleric, Druid, Sorcerer, Wizard | Full level each |
| Paladin, Ranger | Half level (round up) |
| Eldritch Knight, Arcane Trickster | One-third Fighter/Rogue (round down) |

Use the combined level on the multiclass spell slot table in
`08-supplements.md`. Prepare spells **per class** separately. Pact
Magic slots can cast prepared spells from other classes and vice versa.
