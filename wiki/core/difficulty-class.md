# Difficulty Class (DC)

The target number for an [[ability-check]] or [[saving-throw]]. Set by the DM (or by a spell/ability that specifies one). [BR-2024 §07-playing-the-game#Ability Checks]

## Standard DC scale

| Difficulty | DC |
| --- | --- |
| Very easy | 5 |
| Easy | 10 |
| Medium | 15 |
| Hard | 20 |
| Very hard | 25 |
| Nearly impossible | 30 |

[BR-2024 §07-playing-the-game#Ability Checks]

## Spell save DC

A spellcaster's spell save DC is given by the class's spellcasting feature; the general formula is `8 + Proficiency Bonus + spellcasting ability modifier`. See each class's spellcasting page for specifics.
