# Character Origins

**Source:** D&D Beyond Basic Rules (2024)  
**Date:** 2026-05-20 | **Version:** 1.0

---

Origin = **background** + **species**. Together they define formative years and ancestry before your class.

> If using backgrounds or species from older books, ask your DM how to apply 2024 ability score rules and Origin feats.

---

<!-- pagebreak -->

## Background Structure

Each background provides:

| Part | Effect |
| --- | --- |
| Ability Scores | +2 one / +1 another from three listed, or +1 all three |
| Feat | One specified Origin feat |
| Skills | Two skill proficiencies |
| Tool | One tool proficiency |
| Equipment | Package or 50 GP |

## Dragonborn

| Trait | Detail |
| --- | --- |
| Size | Medium (5–7 ft) |
| Speed | 30 ft |
| Draconic Ancestry | Choose dragon type (sets damage) |
