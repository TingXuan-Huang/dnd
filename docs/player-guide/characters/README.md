## Quick-glance threat assessment

**Frontline:** Stannis (only AC 16, only Str-based melee in the party).
**Skirmisher:** Vesper (Sneak Attack — needs an ally adjacent, or surprise).
**Caster — utility/social:** Bard (Inspiration + Persuasion equivalent via Performance).
**Caster — healing/survival:** Druid (Medicine + Nature + spells).

**Party weakness:** Low intelligence and investigation coverage. Mordren will read smarter than them on every check. Bran will not be impressed by their arcana. The party will need to ask questions out loud, not roll for them.
