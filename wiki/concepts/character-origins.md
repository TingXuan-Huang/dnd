# Character Origins

Origin = **background** + **species**. Together they define your formative years and ancestry before your class. [BR-2024 §02-character-origins]

## Background structure

Every background provides:

| Part | Effect |
| --- | --- |
| Ability Scores | +2 to one ability and +1 to another (chosen from three listed), or +1 to each of the three |
| Feat | One specified Origin feat |
| Skills | Two skill proficiencies |
| Tool | One tool proficiency |
| Equipment | A starting package **or** 50 GP |

[BR-2024 §02-character-origins#Background Structure]

## Species structure

| Part | Effect |
| --- | --- |
| Creature Type | All Basic Rules species are Humanoid |
| Size | Medium and/or Small per species |
| Speed | Walking speed in feet |
| Special Traits | Unique abilities |

[BR-2024 §02-character-origins#Species Structure]

## Backwards compatibility

If using backgrounds or species from older books (pre-2024), ask your DM how to apply the 2024 ability score rules and Origin feats. [BR-2024 §02-character-origins]
