# Module 4 — Character Creation (DM Guide)

*What the player guide does **not** cover — what you must check, approve, and track before Session 1*

The [Fallowton character creation guide](../player-guide/fallowton-character-creation.md) teaches the campaign-specific flow. The longer [character creation rules reference](../player-guide/character-creation-guide.md) teaches general 2024 rules. **This document** is your prep checklist: campaign limits, homebrew, balance, secrets, and how player choices hook into Fallowton.

---

## What players handle vs what you handle

| Players do | You do |
|------------|--------|
| Choose class, background, species per 2024 rules | Approve or ban options; offer **Sinkhole Witness** |
| Fill ability scores per your chosen method | Confirm method (standard array recommended for first table) |
| Record HP, AC, skills, equipment on sheet | Verify math; catch illegal armor or wrong spell lists |
| Read [Weave Strain](../campaign/weave-strain-magic.md) for magic | Enforce strain; award **Purification** milestones |
| Use [quick reference](../player-guide/quick-reference-combat-skills.md) at table | Run ruin clock, NPC reactions, consequences |

## Before Session 0 — decisions to lock

### Rules baseline

- [ ] **Books allowed:** 2024 Basic Rules only, or full PHB 2024?  
- [ ] **Starting level:** 1 (recommended) or 3 (subclasses—only if experienced table)  
- [ ] **Ability scores:** Standard array / point buy / rolled (announce before players build)  
- [ ] **Multiclassing:** allowed at level 2+? any restrictions?  
- [ ] **Evil alignments:** usually **no** for heroes  
- [ ] **Feats:** 2024 backgrounds already grant Origin feats—no extra free feat unless you grant it  

### Campaign homebrew (Fallowton)

- [ ] **Weave Strain** on or off? ([weave-strain-magic.md](../campaign/weave-strain-magic.md))  
- [ ] **Cantrips cause strain?** (default: no)  
- [ ] **Magic Initiate / background spells cause strain?** (default: no at level 1, or once/long rest)  
- [ ] **Sinkhole Witness** background allowed? ([background-sinkhole-witness.md](../campaign/background-sinkhole-witness.md))  
- [ ] **Custom backgrounds** from players—require your written OK  

### Tone & safety

- [ ] Lines & veils (horror in ruin, goblin cruelty, village fear)  
- [ ] How graphic is combat at level 1?  
- [ ] PvP? (default no)

## Session 0 — run this checklist live

### 1. Concept fit (each player, one minute)

Ask:

- Why are you in **Fallowton**?  
- Why go down the sinkhole?  
- How do you feel about **magic**? (important for strain and NPCs)  
- Tie to another PC?  

**Red flags to steer:**

- “I steal from the party” with no group buy-in  
- Murderhobo with no village hook  
- Edgelord evil cleric in a superstitious farming village  

**Green lights:**

- Henric’s cousin, Sister Elra’s acolyte, Mordren’s hired guard, sinkhole witness  

### 2. Verify each character sheet (you must check)

For **every** PC, confirm:

#### Identity & source

- [ ] Class from allowed list  
- [ ] Species traits copied (speed, size, darkvision, etc.)  
- [ ] Background = PHB basic four **or** Sinkhole Witness **or** your approved custom  
- [ ] Languages: Common + 2  
- [ ] Alignment acceptable  

#### Ability scores

- [ ] Method matches table agreement  
- [ ] Background boosts applied **after** base array (+2/+1 or three +1s)  
- [ ] No score above 20 at level 1  
- [ ] Primary ability is not dump-statted by mistake (e.g. Wizard with 8 Int)  

#### Proficiencies (stack class + background)

- [ ] **Saving throws:** exactly 2 from class, marked proficient  
- [ ] **Skills:** correct *number* from class + 2 from background, no duplicates unless rules allow  
- [ ] **Tools:** background tool + class tools (e.g. thieves’ tools, instruments)  
- [ ] **Armor training:** only wear armor listed on class (2024: Light/Medium/Heavy/Shields)  
- [ ] **Weapons:** simple/martial as per class  

#### Combat numbers

- [ ] **HP** = class max + Con mod (Barbarian 12+, Fighter 10+, etc.)  
- [ ] **Hit Die** type noted; 1 die at level 1  
- [ ] **AC** calculated with correct armor (no chain mail on Wizard)  
- [ ] **Initiative** = Dex mod  
- [ ] **Attack bonuses** = ability + 2 if proficient  
- [ ] **Damage** includes correct ability mod  

#### Spellcasters only

- [ ] Spellcasting ability correct (see class table below)  
- [ ] **Spell save DC** = 8 + mod + 2  
- [ ] **Spell attack** = mod + 2  
- [ ] Cantrips: correct count  
- [ ] Level 1 spells: prepared vs known rules followed  
- [ ] **Wizard:** spellbook has 6 level-1 spells; spellbook item on sheet  
- [ ] **Warlock:** 2 slots, level 1 slot level; invocations chosen (1 at level 1)  
- [ ] **Cleric/Druid:** prepare list after long rest (Wis mod + level)  
- [ ] Player has **Weave Strain** tracker if using homebrew  

| Class | Spellcasting ability | Level 1 spells note |
|-------|----------------------|---------------------|
| Bard | Charisma | Prepare CHA mod + level |
| Cleric | Wisdom | Prepare WIS mod + level |
| Druid | Wisdom | Prepare WIS mod + level |
| Paladin | Charisma | No level 1 slots until 2 |
| Ranger | Wisdom | No level 1 slots until 2 |
| Sorcerer | Charisma | Know CHA mod + level |
| Warlock | Charisma | Prepare 2; 1 slot |
| Wizard | Intelligence | Prepare INT mod + level; 6 in book |

#### Equipment

- [ ] Took package **or** gold—not double-dipping illegally  
- [ ] Pack contents understood ([equipment-reference](../player-guide/equipment-reference.md))  
- [ ] **Holy symbol / focus** for casters  
- [ ] Ammo (20 arrows) if bow  
- [ ] Starting GP + background GP tracked  
- [ ] One trinket if you allow it  

#### Feats

- [ ] **Origin feat** from background entered (Alert, Magic Initiate, Savage Attacker, Skilled, etc.)  
- [ ] Player knows what the feat does (read feat together)  
- [ ] **Human species** (if used): Versatile grants extra Origin feat—confirm which  

### 3. Party balance (soft check)

Ideal party has:

- Someone who can **take hits** (Fighter, Paladin, Barbarian, Cleric…)  
- Someone who can **find things** (Perception, Investigation)  
- Someone who can **heal** (Cleric, Druid, Paladin at 2)—not mandatory at 1 but helpful  
- **Not** everyone at 6 HP with no armor  

Warn if everyone is a full caster with strain 3 risk in the first dungeon.

### 4. Hook to adventure

Record on your DM sheet:

| PC | Link to Fallowton |
|----|-------------------|
| | Elder’s job / Henric / Mordren / Elra / personal |

Plan **one** scene before descent: village debate ([01-story.md](01-story.md)).

## Custom character requests — how to say yes safely

Players may ask: *“Can my magic work like X?”* or *“Can I have a custom background?”*

| Request type | Your process |
|--------------|--------------|
| **Flavor only** (dreams, accent, superstition) | Yes freely |
| **Mechanical tweak** (strain hurts more, patron is the ruin) | Note on sheet; no extra power |
| **Custom background** | Use Sinkhole Witness as template: 3 abilities, 1 Origin feat, 2 skills, 1 tool, equipment or 50 GP |
| **Extra feat / bonus stat** | Only if whole table gets it |
| **Homebrew class/subclass** | Avoid at level 1 unless you rewrite balance |
| **D&D Beyond homebrew** | Import only after you read every trait |

**Power rule:** If it increases damage, AC, or spell count above same-level peers, say no or tie it to **Purification** milestones.

## Weave Strain — DM enforcement

Players read: [weave-strain-magic.md](../campaign/weave-strain-magic.md)

You track:

| Event | Action |
|-------|--------|
| Leveled spell cast | +1 strain, player picks cost row |
| Strain 3 | No more leveled spells until rest |
| Short rest | −1 strain each |
| Long rest | strain = 0 |
| In ruin + loud magic | Consider +1 ruin clock ([02-interactive-decisions.md](02-interactive-decisions.md)) |
| Milestone | Award Purification I–IV |

**Do not forget:** Healing spells still strain—otherwise Cleric becomes spam with no tension.

**NPC magic:** Mordren also pays strain—or appears to—so PCs see even “experts” suffer.

**Luck & death by build:** Use [Who bears worse luck](../campaign/weave-strain-magic.md#who-bears-worse-luck-and-worse-death-risk) — pick **one** hook per PC (class **or** background). Sorcerer/Warlock +1 Luck debt; Failed Apprentice burnout **DC +2**; Sinkhole Witness worse underground.

Not for session 1, but note now:

- [ ] HP gain (roll or average + Con mod)  
- [ ] New class features / subclass at 3  
- [ ] Proficiency bonus → +3 at level 5  
- [ ] Purification unlocks tied to story  
- [ ] ASI or feat at 4 (2024 class tables)
