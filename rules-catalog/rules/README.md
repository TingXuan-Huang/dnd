# Rules Glossary

Source: [D&D Beyond 2024 Rules Glossary](https://www.dndbeyond.com/sources/dnd/br-2024/rules-glossary#GlossaryConventions)

This folder factors the 2024 rules glossary into focused reference modules. Use [Index by Term](index-by-term.md) when you know the term, or use the module list below when you know the play context.

## Modules

| Module | Use for |
| --- | --- |
| [Conventions](conventions.md) | Glossary reading conventions and general mechanics. |
| [Abbreviations](abbreviations.md) | Short forms such as AC, DC, HP, PB, and XP. |
| [Actions](actions.md) | Action economy and glossary entries tagged as actions. |
| [Conditions](conditions.md) | Condition family entries and their mechanical effects. |
| [Areas of Effect](areas-of-effect.md) | Area-of-effect shape rules and point-of-origin references. |
| [Attitudes](attitudes.md) | Social attitude states used with interaction and Influence checks. |
| [Hazards](hazards.md) | Environmental and ongoing-harm rules tagged as hazards. |
| [D20 Tests](d20-tests.md) | Checks, saves, rolls, proficiency, and other d20-resolution rules. |
| [Combat](combat.md) | Combat positioning, attack consequences, and target-interaction rules. |
| [Movement](movement.md) | Movement modes, speeds, terrain, jumping, swimming, and teleportation. |
| [Senses and Visibility](senses-and-visibility.md) | Light, obscurement, special senses, and related awareness rules. |
| [Damage, Healing, and Rests](damage-healing-rests.md) | Damage, hit points, recovery, rests, and damage modification rules. |
| [Magic and Spells](magic-and-spells.md) | Spellcasting, magical effects, and magic-adjacent glossary entries. |

## Maintenance

- Keep each glossary term canonical in exactly one module.
- Link class, species, and background notes to these local entries for fast lookup.
- Keep external D&D Beyond links for spells, equipment, and source sections that are not part of this local glossary module.
