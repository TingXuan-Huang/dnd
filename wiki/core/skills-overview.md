# Skills Overview

A skill is a narrow specialization within an [[ability-check]]. There are 18 skills; each is governed by a single [[ability-scores|ability]]. [BR-2024 §07-playing-the-game#Skills]

## Proficiency rules

- If you are proficient in a skill used on a check, add your [[proficiency-bonus]] to the roll.
- If you are proficient in both the skill **and** a relevant tool, you have [[advantage-disadvantage|Advantage]] on that check. [BR-2024 §07-playing-the-game#Skills]
- [[expertise]] doubles your Proficiency Bonus for that skill.
