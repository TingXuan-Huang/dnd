# Magic Items A-Z

Magic items are presented in alphabetical order.

If a magic item description capitalizes a creature’s name and presents it in **bold** type, that’s a visual cue pointing you to the creature’s stat block. How to read and use a [stat block](https://www.dndbeyond.com/sources/dnd/free-rules/rules-glossary#StatBlock) is explained in “ [How to Use a Monster](https://www.dndbeyond.com/sources/dnd/br-2024/how-to-use-a-monster) ” and to a lesser degree in the “ [Rules Glossary](https://www.dndbeyond.com/sources/dnd/br-2024/rules-glossary) ”.
