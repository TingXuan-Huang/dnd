# Module 5 — Encounter Balance Reference

This is the DM reference for Session 1 encounter balance. Use [03-tracking-sheets.md](03-tracking-sheets.md) at the table; use this document when preparing, tuning, or explaining why the encounters are built this way.

**Rules baseline:** D&D 2024 / 5.5e Basic Rules and Monster Manual style stat blocks. Prefer official creatures and reskins before custom monsters.

**Party assumption:** 3 player characters, level 1.

---

## Balance Verdict

**Overall verdict:** Pass with tweaks.

The encounter structure is strong: each tunnel has a clear mechanical identity, the threats fit the fiction, and the custom monsters mostly stay close to official low-level play. The main risk is stacking too many level-1 pressures in one delve, especially in Empty Wind.

**Accepted fixes now recorded in Module 3:**

| Area | Fix |
| --- | --- |
| Bessa Hangover | Keep Perception penalty; remove Wisdom saving throw disadvantage |
| Sister Elra | Treat as Acolyte-style village priestess, not full CR 2 Priest |
| Mordren Vale | Spy-style royal field scholar; not a thief, not a session-1 kill target |
| Goblins | Separate Goblin Minion and Goblin Warrior references |
| Grask | Solo Medium fight; add only 1 goblin if warned |
| Edras | Parley-first; no Radiant vulnerability; do not stack with a full skeleton fight |
| Empty Wind | Use Dust Mephits or Breathless Warden as main pressure, not both as full fights |
| Breathless Warden | AC 18, HP 26 as deliberate 3-PC safety adjustment from official HP 33 |
| Silence Aura | Narrative only in Session 1; do not make verbal spells fail |

---

## Global Encounter Dials

| Dial | Session 1 value |
| --- | --- |
| Party | 3 PCs, level 1 |
| Target fight size | One real Medium fight per delve |
| XP target | About 100-200 XP per main fight |
| Enemy HP | Official/default HP unless a safety adjustment is explicitly written |
| Flee rule | On: enemies may flee at half HP when fiction supports it |
| Trap damage | 1d6 remains on; lower to 1d4 only if TPK risk appears |
| Main rule | Do not stack two full Medium fights in the same first-session tunnel |

**Avoid in one delve unless the party is clearly thriving:**

| Avoid | Why |
| --- | --- |
| Edras + 2 Skeletons as one fight | Deadly swing at level 1 |
| Full Skeleton fight, then forced Edras fight | Too much attrition for 3 new level-1 PCs |
| Grask + 2 Goblins | Too many attacks and too much action economy |
| 2 Dust Mephits + Animated Armor as full fights | Two Medium encounters in the same tunnel |
| Spell-fail Silence Aura + Weave Strain | Double-punishes magic characters |

## Surface NPCs

### Bessa Thorn

| Field | Detail |
| --- | --- |
| Role | Bent Plow barmaid; opening social friction; rumor hub |
| Stat block | Commoner |
| Adjustments | DEX 12, CHA 14, Persuasion +4 |
| Gate | Buy a drink for 2 sp, or pass a social check |
| Social checks | Persuasion DC 13, Deception DC 14, Intimidation DC 15 |

**Rumors are intentionally low-value.** They create folk texture and possible wasted motion, not dungeon solutions.

| # | Rumor |
| --- | --- |
| 1 | "Henric's goat stared at the hole and hasn't eaten since." True-ish flavor, no dungeon help. |
| 2 | "Mordren sleeps standing up with one eye open." False. |
| 3 | "The well whispers your name if you listen at dawn." False. |
| 4 | "Old soldiers buried iron under the east field." False. |

**Bent Plow Hangover:** One strong drink is enough. For the first 4 hours of the next adventure day, the PC has disadvantage on Wisdom (Perception) checks and -2 passive Perception. A CON save DC 10 as an action clears it early; retry once per short rest. Do not apply disadvantage to Wisdom saving throws.

**Design reason:** The drink should be funny and inconvenient, not a hidden tax that makes Wind or Blue Light much more lethal.

## Tunnel 1 — Blue Light

**Identity:** Moral undead test. This tunnel asks whether the party treats the dead and the ruin with respect.

### Oath-Bound Dead

| Field | Detail |
| --- | --- |
| Baseline | Skeleton |
| Default package | 2 Skeletons |
| Escalation | 2 Skeletons + 1 Zombie if bodies are looted/desecrated |
| Hard escalation | 3 Skeletons for loud entry or clear desecration |
| Do not use | Shadow in Session 1 |

**Reskin traits:**

| Trait | Effect |
| --- | --- |
| Oath-Slow | If PCs were respectful, undead act last in round 1 |
| Clean Edge | Shortsword only; no bow |
| Blue Echo | On 0 HP, harmless blue flash; clock only rises if desecration track is active |

**Design reason:** The skeletons should signal old duty and danger, not become the whole dungeon day before Edras.

| HP | 27 |
| Speed | 25 ft. |
| Abilities | STR 14, DEX 10, CON 14, INT 10, WIS 14, CHA 12 |
| Immunities | Poison, Exhaustion, Poisoned |
| Resistance | Necrotic |
| Vulnerability | None |
| CR | Treat as 1 |
| Parley | Persuasion DC 13; +2 if respectful |

**Traits and actions:**

| Name | Effect |
| --- | --- |
| Blue Flame Heart | Dim blue light 10 ft.; melee hits deal +1 fire |

**Balance ruling:** Edras can be survived by 3 level-1 PCs if he is parley-first and if the party is not forced to fight him after a full skeleton encounter. If they have already spent resources in Room 2, let Edras test, warn, bargain, or stop at 1 HP to speak.

## Tunnel 2 — Weird Sounds

**Identity:** Goblin tactics, ambush, stolen goods, and a dirty little boss fight.

### Nail-Eater Goblins

| Field | Detail |
| --- | --- |
| Default baseline | Goblin Minion for most guards |
| Tougher baseline | Goblin Warrior only when one goblin should matter |
| Default ambush | 2 Goblin Minions |
| Flee behavior | Flee when bloodied, threatened, or tactically outmatched |

**Goblin references:**

| Creature | Reference |
| --- | --- |
| Goblin Minion | AC 12, HP 7, CR 1/8, Stealth +6, Nimble Escape |
| Goblin Warrior | AC 15, HP 10, CR 1/4, Stealth +6, Nimble Escape |

**Nail-Eater flavor trait:** A goblin can gnaw iron, bone, or coins before threatening someone; if you use mechanics, grant advantage on its next Intimidation attempt against one target. This is not a combat power spike.

**Warn Grask:** If a goblin escapes toward Room 3, Grask is warned and the ruin clock can rise by +1.

---

### Grask the Split-Ear

| Field | Value |
| --- | --- |
| Type | Medium Humanoid (Goblinoid), typically Neutral Evil |
| AC | 16 |
| HP | 22 |
| Speed | 30 ft. |
| Abilities | STR 12, DEX 14, CON 12, INT 10, WIS 10, CHA 8 |
| Saves | DEX +4 |
| Skills | Stealth +6, Intimidation +1 |
| Senses | Darkvision 60 ft., Passive Perception 10 |
| Languages | Common, Goblin |
| CR | About 1 as a mini-boss |

**Actions:**

| Name | Effect |
| --- | --- |
| Hooked Blade | +4 to hit, reach 5 ft.; 1d8+2 slashing; on hit Grask can disengage once per turn |
| Nail Spit | Recharge 5-6; ranged 20 ft.; +4 to hit; 1d4+2 piercing; CON DC 11 or Poisoned until end of Grask's next turn |
| Screech | Bonus action; each allied goblin within 30 ft. can move up to half speed toward Grask's target |

**Balance ruling:** Grask HP 22 is fair as the main Medium fight for 3 level-1 PCs. If warned, add only 1 Goblin Minion or 1 Goblin Warrior. Do not run Grask + 2 goblins in Session 1.

**Background fit:** Hooked Blade and disengage match goblin tactics. Nail Spit fits the Nail-Eater tribe's disgusting iron-chewing identity, but the Poisoned duration must stay short.

## Tunnel 3 — Empty Wind

**Identity:** Hazards, memory loss, silence, and a guardian. This tunnel should feel dangerous through atmosphere and choices, not by stacking every monster at once.

### Archive Wind Hazards

| Hazard | Trigger | Save | Effect |
| --- | --- | --- | --- |
| Wind pressure | First round in Wind Hall; each PC once | WIS DC 12 | Disadvantage on next Perception; heartbeat-like sound |
| Push line | End of round if they stay in hall | DEX or CON DC 11 | Pushed 5 ft. toward Floor That Remembers |
| Memory Slip | Enter Breathless Archive without phrase, touch map early, or clock 3+ | WIS DC 13 | Forget one concrete fact |
| Floor That Remembers | Wrong path or careless crossing | DEX DC 12 | Fail 1d6 piercing; success half |
| Guard prints | Careful investigation | Investigation DC 13 | Follow prints and treat floor crossing as safe |

**Memory Slip limits:**

| Valid forgotten facts | Invalid forgotten facts |
| --- | --- |
| A guard-print path, a trap clue, one NPC name, one surface fact, one phrase fragment | Class features, spell slots, equipment, player sheet mechanics, rules knowledge |

Memory Slip lasts until long rest outside the ruin or Purification I. Lesser Restoration does not fix it. The archive phrase may also end it if the DM wants the puzzle to feel merciful.

**Weave Strain tie-in:** A leveled spell in Wind Hall may require the strain cost to be memory or ill luck. Do not also make the spell fail because of Silence Aura in Session 1.

---

### Dust Mephits

| Field | Detail |
| --- | --- |
| Count | 2 |
| Reference | AC 12, HP 17, CR 1/2, 100 XP each |
| Role | Harass, blind, burst, flee into cracks |
| Tactics | Use dust and breath on clustered PCs; do not chase far |
| Exit rule | If Breathless Warden wakes, mephits flee before armor becomes active |

**Balance ruling:** Two Dust Mephits are already 200 XP, the top of the Medium target for 3 level-1 PCs. Treat them as the main creature pressure if used.

---

### Breathless Warden

| Field | Value |
| --- | --- |
| Baseline | Animated Armor |
| AC | 18 |
| HP | 26 |
| Speed | 25 ft. |
| Attacks | Slam per Basic Rules / Monster Manual |
| Safety note | HP 26 is a deliberate 3-PC adjustment from official HP 33 |

**Traits:**

| Trait | Effect |
| --- | --- |
| Breathless | Immune to Charmed; cannot speak; advantage on saves vs thunder |
| Map-Bound | Dormant until map touched, puzzle failed, or clock stage is high |
| Warden's Duty | Disadvantage on attacks against creatures that did not touch the map |
| Puzzle shutdown | The correct phrase, guard prints, or respectful silence can power it down |
| Silence Aura | Narrative only in Session 1; no spell failure |

**Balance ruling:** Animated Armor AC 18 is frustrating at level 1, so the puzzle solution must be clearly discoverable. If the party already fought Dust Mephits, the Warden should be avoidable, puzzle-solvable, or delayed to a later session.

## Cross-Cutting Answers

| Question | Answer | DM note |
| --- | --- | --- |
| Can 3 level-1 PCs survive Edras HP 27? | Adjust/Yes | Yes if parley-first and not stacked after skeleton attrition |
| Can they survive Grask HP 22 solo? | Yes | Fair as one Medium boss with flee logic |
| Is Hungover too punishing? | Adjusted | Removed Wisdom save disadvantage |
| Does Tamsin weapons-only shop create power creep? | No | No armor/healing/general gear; niche is safe |
| Is Memory Slip + Silence Aura + 2 Mephits + Armor too much? | Yes, too much | Silence Aura made narrative; choose one main Wind pressure |
| Are escalations player-triggered? | Mostly yes | Telegraph desecration, loudness, escaped goblins, map-touching |
| Are the tunnels mechanically distinct? | Yes | Moral undead / goblin tactics / memory-construct hazards |
| Do custom abilities outclass official CR 1/4-1? | Mostly no | Rebuke, Nail Spit, Hooked Blade are okay; spell-fail Aura was too harsh |

## Background And Action Fit

| Character / creature | Fit check |
| --- | --- |
| Bessa Thorn | Persuasion +4 fits an inn rumor gate. Hangover works if framed as strong house ale. |
| Tamsin Bellows | STR 14, INT 13, Smith's tools +4 fits a machine-smith. Weapons-only shop preserves her niche. |
| Mordren Vale | Spy baseline fits only because he has secret royal backing and field training. Describe him as a scholar-agent, not a criminal spy. |
| Elder Bran | Commoner/Noble with villagers arriving in 3 rounds fits village authority. His threat is social, not tactical. |
| Henric Voss | Commoner with pitchfork fits old farmer discoverer. He should not act like an adventuring patron. |
| Sister Elra | Acolyte-style sacred flame once/day fits village chapel authority. Full Priest would be too strong and too grand. |
| Edras | Oath, blue flame, and frighten effect fit an undead guardian. Keep him moral and tragic, not a random boss. |
| Nail-Eater Goblins | Iron-gnawing intimidation and dirty tactics fit cruel scavengers. Keep them nasty, not comic. |
| Grask | Hooked Blade, disengage, and short poison fit a mobile goblin boss. His CHA is low, so Intimidation comes from tribe behavior, not charm. |
| Breathless Warden | Map-Bound armor fits an archive guardian. Puzzle shutdown is important because AC 18 is rough for level 1. |

## Quick Difficulty Dials

| If play feels... | Adjustment |
| --- | --- |
| Too easy | Add 1 Goblin Minion; have Grask Screech early; let Warden wake after a clear warning |
| Too hard | Edras speaks instead of attacking; mephits flee; Grask retreats at half HP |
| Near TPK | Lower needles to 1d4; Edras stops at 1 HP to talk; Warden powers down when phrase is spoken |
| Too confusing | Tell players one clear sensory clue: blue oath, goblin tracks, or wind-heartbeat |
| Too many fights | Convert the next monster into a warning, parley, or retreat scene |

## DM Principle

Session 1 should teach the players that Fallowton's ruin is alive, old, and dangerous. It should not teach them that every curiosity becomes a punishment. Give warnings before escalations, let respectful behavior matter, and let each tunnel have one strong danger rather than many stacked dangers.
