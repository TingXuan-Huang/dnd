# Alignment

A two-axis description of a character's moral and ethical outlook. [BR-2024 §01-creating-a-character#Step 4]

## The nine alignments

| Code | Name | Summary |
| --- | --- | --- |
| LG | Lawful Good | Society's expected good |
| NG | Neutral Good | Best effort within rules |
| CG | Chaotic Good | Conscience over expectations |
| LN | Lawful Neutral | Law, tradition, code |
| N | Neutral | Avoids taking sides |
| CN | Chaotic Neutral | Personal freedom |
| LE | Lawful Evil | Orderly selfishness |
| NE | Neutral Evil | Selfish without qualms |
| CE | Chaotic Evil | Arbitrary violence |

D&D assumes player characters are not evil unless the DM agrees. [BR-2024 §01-creating-a-character#Step 4]
