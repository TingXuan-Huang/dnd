# Ability Check

A [[d20-test]] used when the outcome of an action is uncertain. Named by the ability used (Strength check, Wisdom check, etc.). [BR-2024 §07-playing-the-game#Ability Checks]

## Typical use by ability

| Ability | Typical use |
| --- | --- |
| [[strength]] | Lift, push, break |
| [[dexterity]] | Move quietly, acrobatics |
| [[constitution]] | Endure strain |
| [[intelligence]] | Recall, analyze |
| [[wisdom]] | Notice, read behavior |
| [[charisma]] | Influence, perform |

[BR-2024 §07-playing-the-game#Ability Checks]

## Difficulty Class

The DM sets a [[difficulty-class]] (DC); you succeed if your roll meets or beats it. Standard scale: 5 (very easy) to 30 (nearly impossible).
