# D20 Test

The umbrella term for any roll where you roll a d20 and compare to a target number. There are three kinds: [[ability-check]], [[saving-throw]], and [[attack-roll]]. [BR-2024 §07-playing-the-game#D20 Tests]

## How to make a D20 test

1. Roll a d20.
2. Add the relevant [[ability-scores|ability]] modifier.
3. Add your [[proficiency-bonus]] if you are proficient (with the skill, the save, or the attack).
4. Add or subtract any circumstantial bonuses/penalties.
5. Compare to the [[difficulty-class]] (for checks/saves) or target [[armor-class]] (for attacks). [BR-2024 §07-playing-the-game#D20 Tests]
