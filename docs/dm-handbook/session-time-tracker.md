# Session Time Tracker

A one-page printable sheet for tracking in-game time during a session. Use one per session. Fill in pencil; cross out and update freely.

> **The rule:** Always know what time it is in the world right now. Update the clock whenever the scene changes. Everything else on this sheet supports that single habit.

## 2 · Running clock

Update on every scene change. Cross out the old time, write the new one.

```
Current in-game time:   _______  ·  Day: _______

Scene log (one line per scene change):

[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
[ ____:____ ]  _________________________________________________
```

**Heuristics for advancing the clock:**
- Combat → round to the nearest minute. (4 rounds ≈ 30 sec ≈ "negligible.")
- Searching a room thoroughly → 10 minutes (one dungeon turn).
- Picking a lock, climbing a wall, ID-ing a magic item → 1–10 minutes.
- Short rest → +1 hour.
- Travel within Fallowton → 5–15 minutes on foot.
- Travel to nearby ruin / forest edge → 1–3 hours.
- Long rest → +8 hours.

## 3 · Dungeon-turn tracker (10-min ticks)

Use only when the party is inside a dangerous/timed space (dungeon, ruin, sneaking through somewhere).

**Each hour:** six 10-minute boxes. When all six are filled, **1 hour has passed** — torches burn out, 1-hour spells expire, optionally roll a wandering-monster check.

```
Hour 1  ☐ ☐ ☐ ☐ ☐ ☐    end-of-hour checks: ☐ light  ☐ spells  ☐ encounter
Hour 2  ☐ ☐ ☐ ☐ ☐ ☐    end-of-hour checks: ☐ light  ☐ spells  ☐ encounter
Hour 3  ☐ ☐ ☐ ☐ ☐ ☐    end-of-hour checks: ☐ light  ☐ spells  ☐ encounter
Hour 4  ☐ ☐ ☐ ☐ ☐ ☐    end-of-hour checks: ☐ light  ☐ spells  ☐ encounter
```

**What counts as one turn (10 min):**
- Explore + describe one room
- Search a room carefully
- Force one stuck door
- Disarm one trap
- Move ~120 ft cautiously through corridors
- Listen at a door / parley with something

**Common durations to remember:**

| Spell / effect | Duration |
|---|---|
| Bless, Bane, Hex, Hunter's Mark (initial), Faerie Fire | 1 minute (10 rounds) |
| Mage Armor, Aid, Detect Magic, Find Familiar (session) | 8 hours / 10 min |
| Pass Without Trace, Spike Growth | 1 hour |
| Locate Object, Locate Creature | 1 hour (10 min) |
| Torch | 1 hour of light, 20-ft bright |
| Lamp + 1 flask oil | 6 hours of light |
| Hooded lantern | 6 hours / flask |
