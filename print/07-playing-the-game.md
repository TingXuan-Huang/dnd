## D20 Tests

When success is uncertain, roll d20 + modifiers vs. a target number.

1. Roll d20
2. Add ability modifier
3. Add Proficiency Bonus if proficient (skill, save, or attack)
4. Add circumstantial bonuses/penalties
5. Compare to DC or target AC

Three kinds: **ability checks**, **saving throws**, **attack rolls**.

### Ability Checks

Named by ability used (Strength check, Wisdom check, etc.).

| Ability | Typical use |
| --- | --- |
| Strength | Lift, push, break |
| Dexterity | Move quietly, acrobatics |
| Constitution | Endure strain |
| Intelligence | Recall, analyze |
| Wisdom | Notice, read behavior |
| Charisma | Influence, perform |

**Typical DCs**

| Difficulty | DC |
| --- | --- |
| Very easy | 5 |
| Easy | 10 |
| Medium | 15 |
| Hard | 20 |
| Very hard | 25 |
| Nearly impossible | 30 |

### Saving Throws

Resist hazards (fire, poison, spells). Add Proficiency Bonus if
proficient in that save.

| Ability | Typical save |
| --- | --- |
| Strength | Resist force |
| Dexterity | Dodge area effects |
| Constitution | Endure toxins |
| Intelligence | Disbelieve illusions |
| Wisdom | Resist mental effects |
| Charisma | Assert identity |

### Attack Rolls

Hit if d20 + modifiers ≥ target AC.

| Attack type | Ability |
| --- | --- |
| Melee weapon | Strength |
| Ranged weapon | Dexterity |
| Spell attack | Per spellcasting feature |

- Natural 20: critical hit
- Natural 1: automatic miss

**Base AC:** 10 + Dex modifier (armor/features may replace this).

## Advantage and Disadvantage

Roll two d20s; use higher (Advantage) or lower (Disadvantage). They do
not stack. If you have both, roll one d20.

## Proficiency

Proficiency Bonus by level (character) or CR (monster):

| Level or CR | Bonus |
| --- | --- |
| Up to 4 | +2 |
| 5–8 | +3 |
| 9–12 | +4 |
| 13–16 | +5 |
| 17–20 | +6 |

Apply once per roll. Expertise doubles bonus for that skill/tool check.

proficient in both the skill and a relevant tool, you have Advantage on
that check.

## Rhythm of Play

1. DM describes the scene
2. Players describe character actions
3. DM resolves results (often via D20 Tests)

**Pillars:** social interaction, exploration, combat.
