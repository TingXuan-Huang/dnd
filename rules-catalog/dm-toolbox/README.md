# DM Toolbox

Source: [D&D Beyond 2024 DM’s Toolbox](https://www.dndbeyond.com/sources/dnd/br-2024/dms-toolbox)

This folder factors the 2024 DM Toolbox chapter into focused reference modules. Use [Index by Section](index-by-section.md) when you know the section name, or use the module list below by table workflow.

## Modules

| Module | Use for |
| --- | --- |
| [Creating Backgrounds](creating-backgrounds.md) | Custom background construction for campaign-specific origin facts. |
| [Combat Encounters](combat-encounters.md) | Encounter design, XP budgets, and combat difficulty troubleshooting. |
| [Running Combat](running-combat.md) | Practical table procedures for initiative, hit points, and conditions. |
| [Curses and Magical Contagions](curses-contagions.md) | Curses, demonic possession, and magical contagion examples. |
| [Environmental Effects](environmental-effects.md) | Weather, water, altitude, ice, and other terrain-driven effects. |
| [Fear and Mental Stress](fear-mental-stress.md) | Optional fear and mental stress effects for intense scenes. |
| [Poison](poison.md) | Poison types, harvesting, purchasing, and sample poisons. |
| [Traps](traps.md) | Trap parts, example traps, scaling, detection, and disarming. |
| [Travel Pace](travel-pace.md) | Overland travel pace, terrain limits, extended travel, and vehicles. |

## Maintenance

- Keep DM procedures here and mechanical term definitions in [Rules Glossary](../rules/README.md).
- Keep spell, monster, equipment, and source-section links external unless a local module exists for that exact material.
- Sample poison and trap block headings are local labels restored from the block text for easier lookup.
