# Character Creation

The 5-step process for building a new D&D character. [BR-2024 §01-creating-a-character]

## The five steps

1. **Choose a class** — see `wiki/classes/`. Pick by primary ability and playstyle.
2. **Determine origin** — choose a [[character-origins|background and species]]; pick [[languages]].
3. **Determine ability scores** — see [[ability-score-generation]].
4. **Choose an [[alignment]]**.
5. **Fill in details** — class features, [[hit-points]] at level 1, [[armor-class]], attacks, spells.

## Key derived numbers (Step 5)

| Item | Formula |
| --- | --- |
| Saving throws (proficient) | Ability mod + [[proficiency-bonus]] |
| Skills (proficient) | Ability mod + [[proficiency-bonus]] |
| [[passive-perception]] | 10 + Wis (Perception) modifier |
| [[initiative]] | Dexterity modifier |
| Base [[armor-class]] (unarmored) | 10 + Dex modifier |
| Melee attack | Str mod + proficiency |
| Ranged attack | Dex mod + proficiency |
| Spell save DC | 8 + spellcasting mod + proficiency |
| Spell attack | Spellcasting mod + proficiency |
