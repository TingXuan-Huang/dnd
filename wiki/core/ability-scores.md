# Ability Scores

Every creature has six ability scores: [[strength]], [[dexterity]], [[constitution]], [[intelligence]], [[wisdom]], [[charisma]]. Scores range from 1 to 30; modifiers are derived from the score. [BR-2024 §07-playing-the-game#The Six Abilities]

## Score scale

| Score | Meaning |
| --- | --- |
| 1 | Lowest normal score |
| 2–9 | Weak |
| 10–11 | Human average |
| 12–19 | Strong |
| 20 | Maximum for most adventurers |
| 21–29 | Extraordinary (typically only via magic or epic boons) |
| 30 | Absolute maximum |

[BR-2024 §07-playing-the-game#Ability Scores and Modifiers]

## Score → modifier

| Score | Modifier |
| --- | --- |
| 1 | −5 |
| 2–3 | −4 |
| 4–5 | −3 |
| 6–7 | −2 |
| 8–9 | −1 |
| 10–11 | +0 |
| 12–13 | +1 |
| 14–15 | +2 |
| 16–17 | +3 |
| 18–19 | +4 |
| 20–21 | +5 |

[BR-2024 §07-playing-the-game#Ability Scores and Modifiers]

For scores above 21 (gained via Epic Boons), the modifier continues to scale by +1 per 2 score points: 22–23 → +6, 24–25 → +7, … 30 → +10.

The modifier is added (or subtracted) on every [[d20-test]] using the relevant ability, on damage rolls with the relevant weapon, and on derived values like spell save DCs.
