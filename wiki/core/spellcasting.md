# Spellcasting

The general framework for casting spells. Each spellcasting class has its own variation; this page summarizes the shared elements. [BR-2024 §03-character-classes · BR-2024 §06-spells]

## Class spellcasting summary

| Class | Caster type | Ability | Notes |
|---|---|---|---|
| [[bard]] | Full | [[charisma]] | Prepared; instrument as focus |
| [[cleric]] | Full | [[wisdom]] | Prepared; holy symbol as focus |
| [[druid]] | Full | [[wisdom]] | Prepared; druidic focus |
| [[paladin]] | Half | [[charisma]] | Prepared; holy symbol |
| [[ranger]] | Half | [[wisdom]] | Prepared; druidic focus |
| [[sorcerer]] | Full | [[charisma]] | Known; arcane focus |
| [[warlock]] | [[pact-magic]] (special) | [[charisma]] | Known; arcane focus; short-rest slots |
| [[wizard]] | Full | [[intelligence]] | Spellbook; arcane focus |

## Spell save DC and attack modifier

- **Spell save DC** = 8 + [[proficiency-bonus]] + spellcasting ability modifier
- **Spell attack bonus** = [[proficiency-bonus]] + spellcasting ability modifier
