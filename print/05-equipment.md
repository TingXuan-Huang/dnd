## Weapon Properties

| Property | Effect (summary) |
| --- | --- |
| Ammunition | Ranged attack needs ammo; recover half after fight |
| Finesse | Use Str or Dex for attack and damage |
| Heavy | Disadv. if Str (melee) or Dex (ranged) under 13 |
| Light | Extra Light-weapon attack as Bonus Action |
| Loading | One shot per action/bonus/reaction |
| Range | Normal and long range in feet; Disadv. beyond normal |
| Reach | +5 ft reach |
| Thrown | Ranged attack with melee ability |
| Two-Handed | Two hands to attack |
| Versatile | Two-handed damage in parentheses |

**Proficiency:** Add Proficiency Bonus to attack rolls only with
proficient weapons.

## Weapon Mastery

Each weapon has one mastery property. You must have a feature (e.g.
Weapon Mastery) to use it. Change one mastery on a long rest.

| Mastery | Effect (summary) |
| --- | --- |
| Cleave | Second hit within 5 ft on melee hit (once/turn) |
| Graze | On miss, deal ability-mod damage |
| Nick | Light extra attack as part of Attack action |
| Push | Push Large or smaller 10 ft on hit |
| Sap | Target has Disadv. on next attack |
| Slow | −10 ft Speed until your next turn (max −10) |
| Topple | Con save or Prone |
| Vex | Advantage on next attack vs. same target |
