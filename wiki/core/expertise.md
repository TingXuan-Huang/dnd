# Expertise

When you have Expertise in a skill or tool, your [[proficiency-bonus]] is doubled on any [[ability-check]] using that skill or tool. [BR-2024 §07-playing-the-game#Proficiency]

Expertise is granted by certain class features (notably Rogue and Bard).
