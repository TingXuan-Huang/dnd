## Executive Summary / Core Definition

The **D&D Beyond Basic Rules (2024)** — source slug `br-2024` — is Wizards of the Coast’s free digital rules package for **Dungeons & Dragons fifth edition (2024 revision, often called 5.5e)**. It is the same rules family used on D&D Beyond character sheets and in the 2024 Player’s Handbook line, scoped for zero-cost play.

For **character building**, the source delivers:

- A **five-step creation workflow** (class → origin → ability scores → alignment → sheet details).
- **Twelve character classes**, each with one included subclass in the free rules.
- **Four backgrounds** and **nine species** with 2024-style ability score adjustments and Origin feats.
- A **feat catalog** (Origin, General, Fighting Style, Epic Boon).
- **Equipment** including weapon **mastery properties** and revised armor training.
- **Spellcasting rules** plus full class spell lists in the digital document.
- **Supplement guidance** for multiclassing, starting above level 1, and using options from older books.

> The Basic Rules are a **subset** of the full 2024 PHB. Paid or expanded content adds subclasses, species, backgrounds, and spells not reproduced in this print set.

## Key Challenges / Risks

| Risk | Impact | Mitigation |
| --- | --- | --- |
| **2014 vs 2024 mismatch** | Wrong scores, feats, or species traits if mixing books | Use one rules generation; see `08-supplements.md`. |
| **Incomplete content** | No Battle Master, Light Cleric, etc. in free rules | DM allows PHB purchase content or uses Basic-only options. |
| **Background ASI stacking** | 2024 backgrounds modify scores after array/point-buy | Apply background last; cap 20 at level 1. |
| **Weapon mastery** | Requires class feature to use mastery properties | Note mastery slots on sheet; swap on long rest. |
| **Spell list size** | Analysis paralysis at level 1 | Use class “recommended” spells in class chapter. |
| **Multiclass spell slots** | Easy to mis-count combined slots | Use multiclass table in `08-supplements.md`. |
| **D&D Beyond paywall** | Some spells/items link to locked content | This reference uses only Basic Rules text. |

## Industry / Expert Consensus & Future Outlook

**Consensus:** The 2024 rules are widely treated as an **evolution** of 5e, not a new edition. Character building is **more feat-forward at level 1** (background grants Origin feat; Human gains another). Species use **flat ability score boosts** from background rather than +2/+1 racial bonuses. Martial characters gain **weapon mastery** as a core identity.

**Digital alignment:** D&D Beyond is the de facto standard for sheet automation; `br-2024` is the **free on-ramp** matching paid 2024 PHB data structure.

**Future outlook:**

- More subclasses and species will remain **monetized** on D&D Beyond while Basic Rules stay a limited free tier.
- Tables using **2024 rules** should expect gradual migration from 2014-only content.
- GMs running **2024 Basic + homebrew** should document whether 2014 backgrounds/species are converted (ability rules differ).

> **Takeaway for builders:** Treat `br-2024` as the authoritative free reference for **2024-character-building procedure**; confirm with your DM which books are in play for expanded options.
