# Thieves' Cant

> [[rogue]] feature · Gained at level 1

A secret mix of dialect, jargon, and code that allows rogues to embed hidden messages in normal speech. Only other Rogues (or those taught it) understand. [BR-2024 §03-character-classes#Rogue]
