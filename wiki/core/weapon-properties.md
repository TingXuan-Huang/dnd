# Weapon Properties

A weapon's properties modify how it is used. A single weapon may have multiple. [BR-2024 §05-equipment#Weapon Properties]

## The 10 properties

| Property | Effect (summary) |
| --- | --- |
| [[ammunition]] | Ranged attacks need ammo; recover half after combat |
| [[finesse]] | Use Str OR Dex for attack and damage |
| [[heavy]] | Disadvantage if Str (melee) or Dex (ranged) below 13 |
| [[light]] | Allows extra Light-weapon attack as Bonus Action |
| [[loading]] | One shot per action / bonus action / reaction |
| [[range]] | Normal and long range in feet; Disadv. beyond normal |
| [[reach]] | +5 ft attack reach |
| [[thrown]] | Make a ranged attack using the weapon's melee ability |
| [[two-handed]] | Requires two hands |
| [[versatile]] | Two-handed damage in parentheses |

[BR-2024 §05-equipment#Weapon Properties]

## Proficiency rule

Add your [[proficiency-bonus]] to [[attack-roll]]s only with weapons you are proficient with. [BR-2024 §05-equipment#Weapon Properties]
